Object Oriented Programming and Design

Course Description

This course introduces basic concecpts of OOP (Object-Oriented Programming) with examples in Java, as well as how to design programs with OOP in mind. When finishing this course, you should be able to:

  • understand what is OOP and all related concepts (e.g. object, class, heritage, etc.);
  • write simple Java programs;
  • modeling with UML (Uniform Modeling Language);
  • understand some useful OOP design models (e.g. MVC).
